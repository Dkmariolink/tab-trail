How does Tab Trail work?

Tab Trail automatically tracks your tab navigation as you browse. It captures parent-child relationships when you open new tabs from links, and maintains a complete history of pages visited within each tab.

What's the keyboard shortcut?

Press Alt+T to quickly open the Tab Trail popup and see your current trail.

How do I view the full trail visualization?

Click the "View Full Trail" button in the popup, or right-click the Tab Trail icon and select "Options" to access the explorer.

Can I export my trails?

Yes! Tab Trail supports three export formats:

  • Text - Copy trail as plain text
  • JSON - Download structured data
  • Markdown - Copy formatted trail

Is my data private?

Yes, 100%! Tab Trail stores all data locally on your computer using Chrome's storage API. We never collect, transmit, or have access to your browsing data. Your trails never leave your device.

How much data does Tab Trail store?

By default, Tab Trail keeps data for 7 days. You can adjust this in Settings, or clear data manually at any time. The extension uses Chrome's local storage which has a 5MB limit.

üêõ Troubleshooting

Tab Trail isn't tracking my tabs

  1. Make sure tracking is enabled in Settings
  2. Reload the extension (chrome://extensions)
  3. Check that you've granted all required permissions
  4. Some Chrome internal pages (like settings) cannot be tracked

The explorer page is blank

Try refreshing the page. If it persists, check the browser console for errors and report them to us.
